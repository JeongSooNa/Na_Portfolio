Single Page 영주

Single Page 영주는 경상북도 영주시응 소개하면서 Full Stac Developer가 되기에 앞서 Front Web Developer로서의 역량을 키우기 위해 HTML과 CSS를 이용하여 진행한 프로젝트 입니다.
개발기간은 2022년 5월 약 3일 소요되었고, 깃허브에서 해당 프로젝트 소스를 확인하실 수 있습니다.
깃허브 소스 / YeongJu

Semantic UI를 기초로 HTML 레이아웃을 구성했고, 수업시간에 배운 Flex와 Grid를 중점으로 CSS를 작성했습니다. 미디어 쿼리를 사용하지 못한 점이나 디자인적인 면에서 레이아웃이나 main, personal Color를 정하기 어려웠고 조금 더 동적인 CSS를 구현하기 어렵다는 점이 해당 프로젝트의 보안점 입니다.

Technologies:

  • - CSS3
