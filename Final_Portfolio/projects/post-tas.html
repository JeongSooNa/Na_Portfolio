T.A.S

국가통계포털의 공공 API Data를 사용해 2005년 부터 2020년간 교통사고 데이터를 가져와 사고 및 년, 월 등 유형에 따라 분류하여 시각화를 제공하는 프로젝트입니다.
개발기간은 2022년 6월 28일부터 20022년 7월 10일까지 약 3주 소요되었고, 개발인원은 총 2명(FrontEnd, Javascript, BackEnd, Chart.js)으로 구성되었고, 그 중 백앤드와 차트JS를 활용한 시각화를 담당하여 프로젝트를 진행하였습니다.
MVC패턴을 기반으로 데이터 CRUD 및 유형별 조회를 하였고, Chart.js와 KakaoMap_developer를 사용하여 자료 시각화를 하였습니다.
Controller와 JSP를 사용하여 페이지를 구현하였고 RestController를 사용하여 시각화 및 Data 가져오기는 Ajax로
사용자가 검색, 조회 시 쉽게 알아볼 수 있도록 만들었습니다.

** 프로젝트 개요
KOSIS(국가통계포털)의 교통사고 API를 바탕으로 법규위반별, 사고유형별, 요일 및 도로유형별, 음주유형별, 지역별 카테고리를 나누어
유형별 데이터를 확인하고, 그래프, 지도API등을 통한 시각화자료 확인, Excel 다운로드가 가능합니다.
** 느낀점 7개월간의 교육과정에서 배운 프론트, 백엔드 기능을 사용하면서 실제 사용자가 데이터를 확인하고 사이트를 사용하기에 부족함이 없도록
최대한 상세한 기능을 구현하고 싶었습니다.
팀프로젝트에 있어서 협업이 얼마나 중요한지 알 수 있었고, 프로젝트 진행 중 발생한 깃 Pull/Push간 오류, Brunch 등 많은 오류를 만나면서
소통하고, 해결할 수 있는 능력이 생겼습니다.
** 보안점
백앤드와 기능을 중점으로 프로젝트를 진행하다보니 디자인적인 면이나 프론트에 아쉬운점이 많았습니다.
Excel 다운로드 시 사용자가 선택한 유형별로 다운로드가 될 수 있도록 기능 추가도 필요하며,
관리자 페이지를 조금 더 보완할 필요성을 느꼈습니다.

해당 게시판 소스는 GitHub에서 확인하실 수 있습니다.
