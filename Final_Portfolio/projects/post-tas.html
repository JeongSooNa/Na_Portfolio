국가통계포털의 공공 API Data를 사용해 2005년 부터 2020년간 교통사고 데이터를 가져와 사고 및 년, 월 등 유형에 따라 분류하여 시각화를 제공하는 프로젝트입니다.
개발기간은 2022년 6월 28일부터 20022년 7월 10일까지 약 3주 소요되었고, 개발인원은 총 2명(FrontEnd, Javascript, BackEnd, Chart.js)으로 구성되었고, 그 중 백앤드와 차트JS를 활용한 시각화를 담당하여 프로젝트를 진행하였습니다.

7개월간의 교육과정에서 배운 프론트, 백엔드 기능을 사용하면서 실제 사용자가 데이터를 확인하고 사이트를 사용하기에 부족함이 없도록
최대한 상세한 기능을 구현하고 싶었습니다.
팀프로젝트에 있어서 협업이 얼마나 중요한지 알 수 있었고, 프로젝트를 git organization을 이용해 진행했는데, 이때 commit 충돌이슈가 있었습니다.
이를통해 백업의 중요성을 느끼고, git사용법에 대해 좀 더 공부하는 계기가 되었습니다.

백앤드와 기능을 중점으로 프로젝트를 진행하다보니 디자인적인 면이나 반응형 웹으로 구현하지 못한 점에 아쉬운점이 많았습니다.
